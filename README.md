# Knights Adventure 

**Knights Adventure** is a 2D RPG with an AAA-inspired approach to architecture and system design, developed using **Python 3.14** and **Pygame**.  
The project is focused on the Windows platform and is designed as a scalable game featuring multiplayer support, procedural world generation, and complex RPG mechanics.

---

## Vision

The goal of this project is to explore and implement:

- Scalable 2D game architecture
- Procedural world generation
- Networked multiplayer systems
- Complex RPG mechanics (skills, quests, dialogue systems)
- Performance optimization in a Python-based environment

This project is developed as an **engineering-focused pet project / portfolio project**.

---

## Core Features

### Gameplay & Systems
- **Character class selection**
- **Skill tree system**
- **Inventory and item system**
- **Quest system**
- **Dialogue system**
- **Story-driven narrative with key characters (the King and others)**

### World & Environment
- **Procedural world generation**
- **Dynamic generation of villages and cities**
- **Dynamic and changing backgrounds**
- **Environmental and decorative generation**
- **NPCs: civilians, monsters, merchants**
- **City-based shops**

### Multiplayer
- **Multiplayer mode**
- **Game state synchronization**
- **Support for multiple players in a shared world**

## Getting Started (Windows)

### Requirements
- Windows 10 / 11

### Installation
1. Download and install `tfsetup.exe`
2. Launch `Knights_Adventure.exe`
